## Cборка

    make build
    cd bin

## Использование

 Инициализировать базу данных

    ./adminclient init

Загрузить задачи в базу

     ./adminclient add-problems ../example_problems

Запустить бота

    ./tgbot run

Начать раунд и разослать задачи участникам

    ./tgbot start-round
